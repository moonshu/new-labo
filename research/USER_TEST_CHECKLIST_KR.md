# 사용자 테스트 체크리스트

체크는 `완료/실패/보류` 중 하나로 기록하세요.

## A. 기본 입력/저장

1. [ ] 생각 입력 후 분석 버튼 동작 확인
2. [ ] 질문/유보 상태 저장 성공
3. [ ] 정리 상태에서 근거 추가 후 저장 성공
4. [ ] 정리 상태에서 근거 없이 저장 차단 확인

## B. 탐색/필터

1. [ ] 검색창 키워드 필터 동작
2. [ ] 상태 필터(`질문/유보/정리`) 동작
3. [ ] 운영 필터(`근거 경고/AI 생성`) 동작
4. [ ] 필터 초기화 동작

## C. 노드 상세/편집

1. [ ] 타임라인 노드 클릭 시 우측 패널 열림
2. [ ] 노드 본문 수정 저장 성공
3. [ ] 우측 패널에서 근거 추가/삭제 동작
4. [ ] 경고 상황에서 `유보 상태로 전환` 동작

## D. 사이드바/문제 관리

1. [ ] 문제 추가/이름변경/삭제 동작
2. [ ] 데이터 도구 열기/닫기 동작
3. [ ] JSON 내보내기 동작
4. [ ] JSON 가져오기 동작

## E. 사용성/화면

1. [ ] 페이지 세로 스크롤이 막히지 않음
2. [ ] 작은 화면에서도 버튼/입력이 눌림
3. [ ] `지표 보기` 토글 동작
4. [ ] `용어 쉬운 설명` 토글 동작
5. [ ] 라이트/다크 전환 동작

## F. 데이터 유지

1. [ ] 저장 후 새로고침해도 데이터 유지
2. [ ] 문제별 화면(`/problem/[id]`) 이동 정상

2. 재현 단계:
3. 기대 결과:
4. 실제 결과:
5. 심각도(높음/중간/낮음):
